Alius 6502 - It's Open Source

Every Thing is Open Source

Four Open Source Projects

The whole of the Alius 6502 project is relased to the world as four GitHub projects. We welcome input from anyone. Improve the KiCad project? Fix spelling in the book? Create new demo programs?

The Hardware

The PCB design is a KiCad project published on GitHub. you can download the just the Gerber files and get a PCB printed.

The Software

All the ROM code and the demo programs are on a Github project.

The Book

The user guide is a book written in LaTeX and can be downloaded as a PDF from the GitHub page.

The Website

Even this very website is a GitHub project.
